# Change Request (CR) Flow

This document describes a **generic Change Request governance model**.

## CR Triggers
- Scope change
- Regulatory change
- Business enhancement

## CR Lifecycle
1. Request Submission
2. Impact Analysis
3. Approval
4. Implementation
5. Closure

## BA Responsibilities
- Clarify business impact
- Support estimation
- Update documentation

## Disclaimer
This flow is illustrative and not tied to any organization.
